# match-2-game
Match 2 game implementation with various powerups and levels.

CREATING LEVELS:
You can create custom levels from "Scripts > UI > LevelSelectionView"

CONTROLS:
While playing, press ESC key or settings icon to go back to level selection screen.

USED FRAMEWORKS:
DoTween (https://dotween.demigiant.com/) for animations.
Reflex (https://github.com/gustavopsantos/Reflex) for dependency injection.
